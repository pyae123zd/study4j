# java个人学习项目

本项目记录一些个人学习java的总结和项目实践,作为自己的技术积累。

该项目主要用maven管理。

## 项目结构
```
study4j/(父工程)
  |--study4j-base/(java基础学习记录)
     |--algorithm/(算法学习)
     |--structure/(数据结构学习)
        |--basetype/(基本数据类型学习)
        |--container/(容器类学习)
     |--io/(java io流学习)
     |--nio/(java nio学习)
  |--study4j-utils/(常用工具类封装)
  |--study4j-spider/(个人爬虫项目TODO)
```
